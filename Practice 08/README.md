### Problem Statement:

Design a program that can prompt the user to input two numbers, keep calculating the greatest common divisor until one of the number input is 999.

## Example:
```
Input m: 7
Input n: 49
最大公因數為: 7
Input m: 64
Input n: 128
最大公因數為: 64
Input m: 15
Input n: 10
最大公因數為: 5
Input m: 999
```
